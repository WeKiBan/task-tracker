## Packages
zustand | State management for local persistence
nanoid | Generating unique IDs for local items
@dnd-kit/core | Drag and drop core
@dnd-kit/sortable | Sortable lists
@dnd-kit/utilities | Utilities for DnD
clsx | Class name utility (often paired with tailwind-merge)
tailwind-merge | Class name merging

## Notes
This is a local-only application.
Data is persisted to localStorage.
No backend API calls should be made.
Implements a custom "iOS-inspired" developer tool aesthetic.
